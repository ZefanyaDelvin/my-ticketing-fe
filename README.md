# My Ticketing (React + Vite)

My Ticketing is a web-based ticketing system built using React and Vite. The application allows users to create, manage, and track support tickets through an intuitive interface.

---

## Installation and Setup

### 1. Clone repository

```
git clone https://github.com/ZefanyaDelvin/my-ticketing-fe.git
cd my-ticketing-fe
```

### 2. Install dependencies

```
npm install
```

### 3. Configure environment variables

Create a `.env` file in the project root and add the following:

```
VITE_BASE_URL=http://localhost:5000
```

Adjust based on your backend API URL.

### 4. Run development server

```
npm run dev
```

### 5. Build for production

```
npm run build
```

### 6. Preview production build

```
npm run preview
```
